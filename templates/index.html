About the Teleplexity Engine

⏱

Non-Blocking Sampling

A dedicated CPU-sampling daemon thread fires every 500 ms. The solver loop never waits for psutil reads.

📊

5 Graph Types

Completion %, CPU Usage, Iterations, Remaining Time Prediction, and AES vs Time-Lock comparison.

🔀

Three-Thread Design

Solver thread → CPU sampler daemon → graph renderer daemon. Flask workers are never blocked at any stage.
